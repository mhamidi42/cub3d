# cub3D

## Description
cub3D est un projet graphique réalisé en C dans le cadre du cursus 42.  
Le programme génère une vue en pseudo-3D d’un labyrinthe à partir d’une carte 2D en utilisant la technique du raycasting, inspirée du jeu Wolfenstein 3D.

Le joueur peut se déplacer dans le labyrinthe et observer l’environnement depuis une perspective à la première personne.

## Objectif
L’objectif du projet est d’apprendre à :
- manipuler la bibliothèque graphique MiniLibX
- implémenter un moteur de rendu simple avec le raycasting
- gérer des événements clavier et une fenêtre graphique
- parser un fichier de configuration décrivant une scène
- structurer un projet C plus complexe

## Technologies
- C
- MiniLibX
- Makefile
- Linux

## Compilation
make
./cub3D map.cub
